# Icons

Place your extension icons here:

- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

You can generate icons using any image editor or online icon generator. The icons should represent the SwagDocs extension (e.g., a document with API symbols or similar).

For now, you can use placeholder images or create simple icons programmatically.
